Question-1: What are the different ways to select an element in the DOM?

1.getElementsByTagName নির্দিষ্ট tag নামের সব এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়
2.getElementsByClassName – নির্দিষ্ট class নামের সব এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়.
3.getElementById – id দিয়ে একটি নির্দিষ্ট এলিমেন্ট সিলেক্ট করতে ব্যবহার করা হয়.
4.querySelector – CSS সিলেক্টরের মাধ্যমে প্রথম ম্যাচিং এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়।
5. querySelectorAll – CSS সিলেক্টরের মাধ্যমে সব ম্যাচিং এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয় ।
6.getElementsByName – name অ্যাট্রিবিউট দিয়ে এলিমেন্ট সিলেক্ট করতে ব্যবহৃত হয়

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, এবং textContent এর পার্থক্য সংক্ষেপে
1. innerHTML → HTML ট্যাগ সহ কন্টেন্ট নেয় বা সেট করে। এটি HTML প্রসেস করে, তাই XSS নিরাপত্তা ঝুঁকি থাকতে পারে।
2. innerText → শুধুমাত্র দৃশ্যমান টেক্সট নেয়, লুকানো (display: none) কন্টেন্ট উপেক্ষা করে এবং অতিরিক্ত স্পেস বাদ দেয়।
3. textContent → Hidden টেক্সট সহ সম্পূর্ণ টেক্সট নেয়, কিন্তু HTML প্রসেস করে না। এটি স্পেস এবং লাইন ব্রেক সংরক্ষণ করে।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি বড় এলিমেন্টের মাধ্যমে ছোট এলিমেন্টের event হ্যান্ডেল করা।
এর ফলে আলাদা আলাদা ইভেন্ট লিসেনার না লাগিয়ে একটিই parent লিসেনার ব্যবহার করা যায়, যা পারফরম্যান্স বাড়ায়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো এমন একটি প্রক্রিয়া যেখানে কোনো child element-এ ইভেন্ট ট্রিগার হলে, সেই ইভেন্টটি ধাপে ধাপে তার parent, তারপর grandparent হয়ে document পর্যন্ত পৌঁছে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

1.আমরা document.createElement("tagName") ব্যবহার করে নতুন element তৈরি করতে পারি।
2.appendChild() → Parent-এর শেষে যোগ করা
3. append() → Multiple বা Text যোগ করতে পারে
4. prepend() → Parent-এর শুরুতে যোগ করা
5.insertBefore() → নির্দিষ্ট জায়গায় যোগ করা
6.remove() → সরাসরি element মুছতে পারে
7.removeChild() → Parent থেকে child element মুছা
